Edit Details:

Each tenant and person considered as one record and one row in a table similar to Excel row Details page does allow Add/Delete records and Edit existing Each record has following fields: name(required) and unit. As additional info: Phone, Email and Website As advanced each record can display HTML formatted text and images. Normally advanced editor is hidden and opens only if record contains HTML formated text. With a click on “Edit button” text turned on in editable area bringing 24 most common editing tools. in Addition to 19 text formatting tools there are Insert Image, Insert/Edit table, Remove formatting, Horizontal line, HTML direct edit. Editor supports copy / paste function from HTML browsers.
